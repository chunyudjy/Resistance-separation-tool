
### 📅 项目推进时间表 (Schedule)

#### **5月下旬：方案设计与基准（Baseline）建立**

* **核心任务**：确立实验矩阵，明确扣电（Coincell）与单层软包（Pouch cell）的制样标准；在 Python 中搭建好数据解析的底层框架。
* **具体行动**：
1. **材料与电池制备**：安排单层涂布（对照组）与双层涂布（实验组）的电极打样。确保**总厚度、总面密度、总压实密度完全一致**。
2. **测试夹具与环境准备**：检修 Solartron 和 BioLogic 设备，统一测试温度（恒温槽 $25^\circ\text{C}$ 和 $45^\circ\text{C}$，温度对 $R_{ct}$ 和 $R_{ion}$ 影响极大）。
3. **Python 脚本初版**：用 VSCode 编写好等效电路（ECM）拟合脚本（基于 `scipy.optimize` 或 `galvani` 等库），用于后续批量处理 EIS 和 HPPC 数据。



#### **6月上旬：扣电（Coincell）多维度测试与机理验证**

* **核心任务**：利用对称电池（Symmetric cell）和半电池，优先攻克 $R_e$（接触电阻）和 $R_{ion}$（孔隙率影响）的定性分离。
* **具体行动**：
1. **组装半电池（对金属锂）**：进行基础的 EIS 测试，观察高频截距和中频圆弧。
2. **组装对称电池（重点！）**：这是分离阻抗的绝招。组装“正极-正极”或“负极-负极”的对称电池，**彻底消除金属锂负极的干扰**。
* 通过对称电池的 EIS，利用传输线模型（TLM）直接提取**涂层内部的离子传输阻抗 $R_{ion}$**。
* 对比高频截距，直接验证双层涂布下层高导电设计是否降低了**集流体与涂层的接触电阻（包含在 $R_e$ 中）**。





#### **6月中旬：单层软包电池（Pouch Cell）制造与首次全电池联测**

* **核心任务**：将扣电上的结论向更接近实际工况的单层软包（全电池）移植，开始 HPPC 与 EIS 的联动。
* **具体行动**：
1. **全电池组装**：制备单层软包全电池。软包的压紧力更均匀，比扣电更能反应真实的 $R_e$。
2. **首次 HPPC+EIS 联测**：
* 在不同 SOC（如 90%, 50%, 10%）下静态平衡后做 **EIS**。
* 紧接着做 **HPPC（大电流脉冲）**。


3. **交叉验证**：对比 BioLogic（高采样率脉冲）与 Neware 的 HPPC 数据，看新威设备在 $10\text{ms}/100\text{ms}$ 时的电压骤降能否与 Solartron 测试的 $R_e$ 对齐。



#### **6月下旬：数据建模与算法对齐（关键节点）**

* **核心任务**：利用 Python 将 HPPC 的时域极化曲线与 EIS 的频域阻抗谱进行数学上的解耦与对齐。
* **具体行动**：
1. **EIS 拟合**：利用 Solartron/BioLogic 自带软件或 Python 库，对全电池 EIS 进行等效电路拟合（$R_e + R_{ct}//C_{dl} + W_{ion}$），解出精确的 $R_e, R_{ct}, R_{ion}$。
2. **HPPC 参数辨识**：用 Python 对软包的 10s 脉冲曲线进行**二阶 RC 模型拟合**。
3. **对齐指标**：验证 HPPC 拟合出的 $R_0$ 是否等于 EIS 的 $R_e$；HPPC 的 $R_1, R_2$ 是否与 EIS 的 $R_{ct}, R_{ion}$ 具有强正相关性。若对齐成功，则证明手法有效。



#### **7月上旬：高功率工况模拟与倍率/温度边界拓展**

* **核心任务**：针对“高功率电芯”定位，验证双层涂布在极端工况（大电流、低温）下的阻抗优势。
* **具体行动**：
1. **倍率 HPPC 测试**：由于是高功率产品，设计 $1\text{C}, 3\text{C}, 5\text{C}, 10\text{C}$ 的梯度脉冲，观察在高电流密度下，双层涂布的 $R_{ct}$ 和 $R_{ion}$ 是否表现出比单层更慢的极化增长。
2. **低温阻抗测试**：在 $0^\circ\text{C}$ 或 $-20^\circ\text{C}$ 下进行 EIS 测试。低温下 $R_{ct}$ 和 $R_{ion}$ 会急剧放大，此时双层涂布的动力学优势（高孔隙率表层）会被放大，最容易拉开与单层的差距。



#### **7月中旬：统计学重复性验证与数据收敛**

* **核心任务**：确保测试手法的重现性，排除电池制样差异（B2B 差异），形成可信的数据报告。
* **具体行动**：
1. **多批次平行测试**：每组（单层 vs 双层）至少取 5-8 支软包电池进行相同的 HPPC+EIS 流程。
2. **数据自动化处理**：运行写好的 Python 脚本，一键生成阻抗随 SOC 变化的趋势对比图（R-SOC 曲线），确保有显著的统计学差异（$p < 0.05$）。



#### **7月下旬：标准手法（SOP）确立与项目结题汇报**

* **核心任务**：固化测试手法，形成公司内部 R&D 标准。
* **具体行动**：
1. **编写 SOP 文档**：明确规定：测试前静置时间、EIS 频率范围（如 $10\text{kHz} \sim 10\text{mHz}$）、HPPC 脉冲电流大小、各阻抗分量的定义与提取数学方法。
2. **成果汇报**：展示双层涂布在 $R_e$（箔材接触优化）和 $R_{ct}/R_{ion}$（梯度孔隙率优化）上分别改善了百分之多少，顺利结题。
